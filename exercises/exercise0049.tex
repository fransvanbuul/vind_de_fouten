\begin{opgave}
Voor je explosie experiment bouw je een mega vuurwerkshow. Je hebt verschillende
types vuurwerk nodig:

\begin{itemize}
\item Sterretjes: 45 stuks a \texteuro{2,50} per stuk
\item Romeinse kaarsen: 28 stuks a \texteuro{6,75} per stuk
\item Grote knallers: 15 stuks a \texteuro{12,30} per stuk
\item Finale vuurwerk: 8 stuks a \texteuro{23,50} per stuk
\end{itemize}

Je krijgt 12\% korting op alles boven \texteuro{500}. Je budget is \texteuro{650}.
Kun je alles kopen? Hoeveel heb je over of tekort?
\end{opgave}

\begin{oplossing}
Kosten per type:
Sterretjes: 45 x \texteuro{2,50} = \texteuro{112,50}
Romeinse kaarsen: 28 x \texteuro{6,75} = \texteuro{189,00}
Grote knallers: 15 x \texteuro{12,30} = \texteuro{184,50}
Finale vuurwerk: 8 x \texteuro{23,50} = \texteuro{188,00}
Totaal: \texteuro{112,50} + \texteuro{189,00} + \texteuro{184,50} + \texteuro{188,00} = \texteuro{674,00}

Korting (12\% omdat boven \texteuro{500}): \texteuro{674,00} x 0,12 = \texteuro{80,88}
Te betalen: \texteuro{674,00} - \texteuro{80,88} = \texteuro{593,12}

Budget: \texteuro{650,00}
Over: \texteuro{650,00} - \texteuro{593,12} = \texteuro{56,88}

Antwoord: \texteuro{56,88} over
\end{oplossing}
