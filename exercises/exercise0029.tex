\begin{opgave}
Voor je natuurkunde project ga je een vulkaan model bouwen met echte chemische
reacties. Je hebt de volgende materialen nodig:

\begin{itemize}
\item Bakingsoda: \texteuro{3,75} per doos
\item Azijn: \texteuro{2,50} per fles
\item Rode voedselkleurstof: \texteuro{4,20} per flesje
\item Klei voor de vulkaan: \texteuro{8,90} per kg
\item Verf: \texteuro{6,30} per potje
\end{itemize}

Je koopt 3 dozen bakingsoda, 5 flessen azijn, 2 flesjes voedselkleurstof, 4 kg klei
en 4 potjes verf. Je budget is \texteuro{95}. Heb je genoeg geld? Hoeveel geld
heb je over of te kort?
\end{opgave}

\begin{oplossing}
Kosten:
Bakingsoda: 3 x \texteuro{3,75} = \texteuro{11,25}
Azijn: 5 x \texteuro{2,50} = \texteuro{12,50}
Voedselkleurstof: 2 x \texteuro{4,20} = \texteuro{8,40}
Klei: 4 x \texteuro{8,90} = \texteuro{35,60}
Verf: 4 x \texteuro{6,30} = \texteuro{25,20}
Totaal: \texteuro{11,25} + \texteuro{12,50} + \texteuro{8,40} + \texteuro{35,60} + \texteuro{25,20} = \texteuro{92,95}

Budget: \texteuro{95,00}
Over: \texteuro{95,00} - \texteuro{92,95} = \texteuro{2,05}

Antwoord: \texteuro{2,05} over
\end{oplossing}
